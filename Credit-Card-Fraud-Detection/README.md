# Credit Card Fraud Detection

## Overview
This project detects fraudulent credit card transactions using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook
- Power BI

## Dataset
The dataset contains credit card transactions labeled as fraudulent or legitimate.
